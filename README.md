# Mandira Path: A Digital Heritage Platform for Indian Temples and Spiritual Knowledge

## Overview

Mandira Path is a web platform dedicated to documenting and presenting the **spiritual, historical, cultural, and architectural significance of Indian temples**. It provides a structured and visually engaging way for students, youth, and culture enthusiasts to explore Indiaâ€™s temple heritage.

## Live Demo

Explore the platform here: [Mandira Path](https://mandira-path-new.netlify.app/)

## Features

* Multi-dimensional categorization of temples by:

  * Geography (North, South, East, West India)
  * Dynasty (Kakatiya, Chola, Vijayanagara, Hoysala, etc.)
  * Deity (Shiva, Vishnu, Shakti, others)
  * Special significance (architectural uniqueness, historical importance)
* Card-based UI for clean, intuitive navigation
* Detailed temple descriptions highlighting spiritual, historical, and architectural significance
* Youth-friendly presentation for easy learning
* Scalable content structure for future enhancements

## Project Development Steps

1. **Requirement Analysis:** Defined goals to create a youth-friendly temple heritage platform.
2. **Content Research & Collection:** Gathered authentic information about temples, including history, architecture, deity, and dynasty.
3. **Information Architecture:** Designed multi-layered categorization and navigation structure.
4. **UI/UX Design:** Created clean, culturally respectful designs using Figma/Canva.
5. **Frontend Development:** Built static pages with reusable HTML, CSS, and JavaScript components.
6. **Content Integration:** Mapped researched temple data into structured UI components.
7. **Testing & Refinement:** Verified responsiveness, content accuracy, and overall user experience.
8. **Deployment & Documentation:** Hosted on Netlify and prepared comprehensive project documentation.
